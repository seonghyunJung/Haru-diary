# Haru-diary

## AI 감성분석 일기 서비스 - [Page Link](http://15.165.121.165/)

### 소개
예전에 진행했던 졸업 프로젝트를 Django를 이용해 웹사이트 형태로 재구성하였습니다. <br/>
일기를 작성하면 AI가 해당 일기의 감정을 분석하여 알려주고, 사용자는 달력 및 통계 기능을 통해 지난 감정 추이를 한눈에 볼 수 있습니다. <br/>
[졸업 프로젝트 링크](https://github.com/CSID-DGU/2022-1-CECD3-FRIDAY-5) <br/>

### 개발동기
졸업 프로젝트에서 팀장 및 인공지능 모델 부분을 맡았었는데 개인적으로 프론트엔드 부분과 백엔드 부분을 더 잘 알고 진행했으면 좋았을 걸 하는 아쉬움이 남았습니다.
최근에 공부하고 있는 Django를 이용해 직접 웹서비스 형태로 구현해 보았습니다.
